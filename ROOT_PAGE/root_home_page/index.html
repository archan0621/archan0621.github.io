About Us

Hacking is open door


저희 동아리는 2012년에 설립되었습니다. 해킹 보안이라는 희소성 있는 분야를 공부해보고 싶은 학생들끼리 만나 동아리를 만들었습니다. 비록 1명에서 시작된 동아리지만 지금은 학교 어떤 동아리보다 화목하고 선후배간이 제일 친근한 분위기를 가진 자유롭고 즐거운 동아리입니다. 어떤 동아리보다 열심히 하고 일반적인 프로그래밍을 하는 학생들 보다 훨씬 더 진보되고 타의 모범이 되는 학생들이 모였습니다. 동아리실에서 꾸준히 공부를 하고 공부한것으로 대회에 나가서 우리가 배운 성과에 대해 모두에게 증명하고 있습니다. 서울디지텍고등학교 어떤 동아리보다 화목하고 즐거운 해킹보안 동아리 Root입니다.

Programming

It 기술의 기초가 되는 프로그래밍을 배웁니다. 동아리에 들어오게 되면 시스템 해킹을 배우는 학생들은 C언어를 배우게 되고 웹 해킹을 배우는 학생들은 HTML을 먼저 배우게 됩니다. 이런 기초적인 언어를 배우면서 해킹의 기초를 다집니다.

System Hacking

시스템 해킹은 시스템의 최고 관리자 권한을 휘득하는 해킹 분야입니다. 프로그램의 취약점을 공부하고 메모리 구조를 배웁니다. 배운 메모리 구조와 취약점을 역 이용하여 시스템을 해킹합니다.

Web Hacking

웹 해킹이란 웹 사이트의 취약점을 이용하여 권한을 얻는 해킹 분야입니다. 웹 해킹의 목표는 웹 서버, 웹 애플리케이션, 웹 사용자를 해킹하는것을 목표로 합니다. 또한 해킹뿐만 아닌 웹 사이트를 만들때 보안 취약점을 방어하는 코딩법도 배웁니다.

Network Hacking

네트워크 해킹이란 네트워크의 취약점을 통한 해킹입니다. 보통 시스템 해킹과 함께 이루어지는 경우가 많은 해킹입니다. 시스템에 악성코드를 넣고 변조된 웹 사이트에 접속하도록 만들거나 서버의 취약점을 이용해 루트 권한을 얻는 등의 방법이 있습니다.

Project

각자 배운 여러가지 보안 기술이나 취약점등을 활용하여 동아리 부원들과 프로젝트를 진행합니다. 꼭 보안 프로젝트가 아니여도 자기가 배운 프로그래밍 언어들로 웹, 앱 프로젝트도 진행 할 수 있습니다. 이렇게 진행한 프로젝트는 모두 자신의 포트폴리오에 넣을 수 있게 됩니다.

CTF

Capture the flag, 한국어로 해석하면 flag(정답)을 잡아라의 약자입니다. 해킹 대회라고도 할 수 있습니다. 동아리에서 배운 해킹 기술이나 보안 취약점들을 활용하여 대회에 나가여 더 다양한 보안 기법들과 문제들을 경험하며 자신의 실력을 쌓을 수 있습니다.
